## Folder Structure

　Flutterフォルダの構成は省略
　lib以下の構成説明
　
　lib - components(再利用可能な画面構成パーツ置き場)
　    - controllers(通信系などの画面処理以外の機能)
　    - maplat(maplat固有のデータ置き場)
　    - model(データ構造定義)
　    - view(画面)
　    　- top(トップ画面)
　    　- map(map画面)
　    　- meguri(巡り画面)
　    - main.dart(アプリエンドポイント)
